# 🌌 Planets - Sony SRD × Unity 3D Visualization Project

### 🎓 Tokyo Design Technology Center College (TECH.C)
Super AI Creator Course  
Project by: 山端修太郎（3年）  
Team Members: AI / CG / Game 専攻混成チーム  

---

## 🪐 概要
このプロジェクトは **Sony Spatial Reality Display（SRD）** を活用し、  
惑星・衛星・ロケットなどの立体的な3Dモデルをリアル空間に表示・観察できるアプリケーションです。  
研究・教育・エンタメ分野で新たな価値を提供することを目的としています。

---

## 🧩 主な機能
- 🛰️ **Sony SRDカメラトラッキング対応**
- 🌍 **惑星モデルの3D回転・拡大縮小**
- 🚀 **ロケットなどの3Dオブジェクト投影**
- 🎮 **Unity C#によるインタラクション制御**
- 🔭 **教育・研究向けビジュアライゼーション**
